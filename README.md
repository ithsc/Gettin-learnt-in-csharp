#### Exercises/InputOutput

**UserProps.cs**

Be användaren att svara på dessa frågor:

Vad är ditt namn? 
Vad är din ålder? 
Vad är din längd? (cm) 
Vart bor du? 
Vad är din hobby? 
Vad är din superkraft? 

**Output**

```
Sammanfattning:
Namn: 
Ålder: 
Längd: 
Ort: 
Hobby: 
Superkraft: 
```
